= AWS Simple DB

== Requires ..

* {aws_sdb gem}[http://rubygems.org/gems/aws-sdb] (not included).
* {aws_sdb_proxy plugin}[http://github.com/bansalakhil/aws_sdb_proxy] (not included).

==

* rake aws_sdb:start_proxy
* script/console
* Post.create(:title => 'My first SimpleDB post')
* Post.find(:all)
* rake aws_sdb:stop_proxy
